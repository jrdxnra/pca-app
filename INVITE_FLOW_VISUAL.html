Admin Responsibilities

Admin Panel Actions

  • Create invites (email + role).
  • View pending invitations.
  • Revoke invites if needed.
  • Remove coaches to disable access.

Admin Data Effects

  • Creates invitations with status: pending.
  • Membership is created only after acceptance.
  • Roles enforce permissions in Firestore rules.

Coach Responsibilities

Coach Actions

  • Open invite link.
  • Sign in with Google.
  • Accept invite.
  • Complete calendar connection.

Access Gates

Calendar connect required

Until the calendar is connected, the coach can not access the main app.
