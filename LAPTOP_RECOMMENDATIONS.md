# Laptop Recommendations for Next.js Development

## Required Specs

**Minimum:**
- CPU: Intel i5 (11th gen+) or AMD Ryzen 5 (5000 series+)
- RAM: 16GB (8GB is too tight for dev server)
- Storage: 256GB SSD minimum (512GB recommended)
- Budget: $800-1,200

**Ideal:**
- CPU: Intel i7 or AMD Ryzen 7
- RAM: 16GB (32GB if you can afford it)
- Storage: 512GB+ SSD
- Display: 14-15" (good for coding)

## What to Look For

### CPU Priority:
1. **AMD Ryzen 5 7530U/7535U** - Great performance, good battery
2. **Intel i5-1235U/1335U** - Good performance, reliable
3. **AMD Ryzen 7 7730U** - Better if you can afford it
4. **Intel i7-1255U/1355U** - Better if you can afford it

### RAM:
- **16GB is essential** - Don't settle for 8GB
- Check if RAM is upgradeable (some laptops have soldered RAM)

### Storage:
- **SSD is mandatory** - No HDDs
- 256GB minimum, 512GB recommended
- NVMe SSD is faster than SATA SSD

### Display:
- 14-15" is good for coding
- 1080p (1920x1080) is fine
- Higher resolution is nice but not necessary

## Red Flags to Avoid

- ❌ **8GB RAM** - Not enough for dev server
- ❌ **HDD storage** - Too slow
- ❌ **Celeron/Pentium CPU** - Too weak
- ❌ **4GB RAM** - Absolutely not enough
- ❌ **No SSD** - Will be painfully slow
