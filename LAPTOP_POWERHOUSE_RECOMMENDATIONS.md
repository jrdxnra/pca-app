## Key Specs to Look For

### CPU (Priority):
1. **AMD Ryzen 7 7840U/7840HS** - 8 cores, 16 threads, excellent
2. **AMD Ryzen 9 7940HS** - Even better if budget allows
3. **Intel i7-13700H/13800H** - 14 cores, excellent performance
4. **Intel i9-13900H** - Top tier if budget allows

### RAM (Critical):
- **32GB is the sweet spot** - Will never be a bottleneck
- 16GB works but 32GB gives you headroom for:
  - Dev server (2-3GB)
  - Browser with 20+ tabs (4-6GB)
  - VS Code + extensions (1-2GB)
  - OS + background apps (2-3GB)
  - Still have 15GB+ free!

### Storage:
- **1TB NVMe SSD** - Fast and plenty of space
- NVMe is 3-5x faster than SATA SSD

### Display:
- **15-16"** - More screen = better for coding
- **1920x1200 or higher** - Good resolution
- 4K is nice but not necessary
